# Portafolio Sara Correa 🌸

¡Bienvenida/o al portafolio web de Sara Correa! Un sitio personal, creativo y lleno de detalles kawaii, hecho con mucho amor pastel.

## ✨ Características
- **Diseño pastel y aesthetic**: Colores suaves, decoraciones y animaciones.
- **Secciones**: Sobre mí, Experiencia, Proyectos, Habilidades, Contacto.
- **Animaciones**: Efectos de entrada, hover, shimmer y tilt en tarjetas.
- **Asistente virtual princesa**: Chatbot animado, inspirado en la Princesa Peach, que responde preguntas, cuenta chistes y te ayuda a navegar.
- **Responsive**: Se adapta a dispositivos móviles y escritorio.

## 📁 Estructura del proyecto
```
├── index.html         # Página principal
├── styles.css         # Estilos y animaciones
├── script.js          # Lógica JS y chatbot
├── profile.jpg        # Foto de perfil
├── instagram.svg      # Icono Instagram
├── tiktok.svg         # Icono TikTok
├── github.svg         # Icono GitHub
├── ...
```

## 🚀 ¿Cómo usar este portafolio?
1. **Descarga o clona este repositorio**
2. Abre `index.html` en tu navegador favorito
3. ¡Listo! Puedes personalizar textos, imágenes y colores editando los archivos correspondientes.

## 🤖 Asistente Virtual Princesa
- Haz clic en el ícono 💬 para abrir el chat.
- Puedes preguntarle sobre proyectos, experiencia, consejos, pedir chistes, emojis y más.
- Si pides "otro chiste", te contará uno diferente cada vez.
- Inspirada en la Princesa Peach, con animaciones y respuestas kawaii.

## 🛠️ Tecnologías usadas
- HTML5 + CSS3 (Flexbox, animaciones, media queries)
- JavaScript (vanilla, sin frameworks)
- SVG para avatar y decoraciones
- [AOS](https://michalsnik.github.io/aos/) para animaciones al hacer scroll

## 🎨 Personalización
- Cambia los colores en `styles.css` (variables al inicio)
- Edita el avatar princesa en el SVG dentro de `index.html`
- Agrega tus propios proyectos, habilidades y experiencia
- Modifica o amplía las respuestas del chatbot en `script.js`
